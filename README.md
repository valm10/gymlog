## Environment

- Create .env file
  EXPO_PUBLIC_SUPABASE_URL=https://YOUR-PROJECT.supabase.co
  EXPO_PUBLIC_SUPABASE_ANON_KEY=YOUR-ANON-KEY

---

## QuickStart

npm install
cp .env.example .env # fill Supabase URL + anon key
npm run start # or: npm run ios / npm run android
